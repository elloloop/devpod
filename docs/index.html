Features

Everything you need to ship.

Four verbs replace your entire workflow. No branches, no rebasing, no merge conflicts you didn't cause.

No Git Required

Start features, save diffs, sync, ship. No branches, commits, rebases, or push/pull in your vocabulary.

Stacked Diffs

Break features into small reviewable pieces. Edit any diff in the stack — the rest rebase automatically.

Local CI Runner

Run your .github/workflows/ locally. Same YAML, same actions, same results. No pushing to test.

VS Code Diff Viewer

Monaco Editor in the browser. Syntax highlighting, side-by-side, dark and light themes.

Version History

Every change is a snapshot. Compare any version of any diff. True interdiff between review rounds.

One Binary

6.6MB Go binary. 20ms startup. No Node.js, no Docker, no dependencies. Works offline.

Workflow

Four steps. That's the whole workflow.
